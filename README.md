# workDayScheduler

# Description 
Work Day Scheduler allows a user to create a schedule of their 9-6pm work day. If the hour has past, the area will be highlighted in gray, the red will indicate the current hour, and the green indicates the hours that have yet to pass. You may type in a title for the hour and save it to the local storage as well be pressing the Save Button. 

# Technologies Used
